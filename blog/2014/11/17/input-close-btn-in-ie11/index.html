去掉IE11中input右侧的关闭叉叉

作者: Barret李靖 2014-11-17 00:00:00 本文发布时间为2014年11月17日00时00分00秒 分类: 前端杂烩 标签: CSS 下面是正文内容 评论数: 热度:

原文地址：http://www.barretlee.com/blog/2014/11/17/input-close-btn-in-ie11/，参与评论

在 IE11 下，浏览器自作多情在 text input 组件上加一个 close 叉叉：

这么整：

input::-ms-clear { display: none; }

这种伪元素类似于 web component.
